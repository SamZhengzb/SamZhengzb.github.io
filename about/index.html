项目经历

面向低功耗智能识别算法的 SoC 设计

一种高能效单目标检测网络加速器设计，完成基于 RISC-V 的 SoC 仿真验证、综合等前端设计流程

  • 负责高能效加速器的架构设计、RTL 仿真验证以及 FPGA 验证
  • 基于蜂鸟 e203 RISC-V core 和 AHB BUS 搭建 SoC, 负责算法的硬件部署和 C 驱动程序设计
  • 负责 SoC 的 RTL 仿真验证、DC 综合、形式验证等前端设计流程

面向辅助驾驶的智能硬件加速平台设计与实现

研究面向车辆周边环境信息感知的深度学习加速器与配套软件库，设计一种支持多种车辆环境感知算法的可配置硬件加速器，并基于 Xilinx ZYNQ 平台完成 SoC 的搭建与系统实现[demo展示]

  • 设计图像分块调度策略，优化片上特征图缓存存储
  • 基于 Xilinx ZYNQ-7000 平台搭建 SoC 系统，负责 AXI-Stream 接口的 RTL 设计与验证
  • 利用 Xilinx SDK 工具开发深度学习加速器的 C 驱动程序，完成算法的硬件部署
  • 效果：MobileNetV1+YoloV3 tiny 模型在加速器上部署，目标检测网络的推理速度为 52 fps

2021 DAC System Design Contest

在 Xilinx Ultra96v2 平台上实现高速、高准确率且低功耗的飞行器高空目标检测深度学习算法，完成
高能效的硬件加速器设计与目标检测算法的硬件部署 [demo展示]

  • 负责高能效加速器的架构设计与 RTL 仿真验证
  • 基于 Xilinx UltraScale+ 平台搭建 SoC 系统，负责 AXI-Stream 接口的 RTL 设计与验证
  • 利用 PYNQ 软件框架开发深度学习加速器的 Python 驱动程序，完成算法的硬件部署
  • 效果：官方测试集 IoU 为 0.68，检测速度为 158 fps，功耗为 4.62W，能效为 34.2 Image/J
